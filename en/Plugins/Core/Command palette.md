### Usage tips

- To see a list of all commands and shortcuts via the Command palette, don't type anything into the prompt. Use arrow keys to move up and down the list, and hit `Enter` to activate the command. 

- Search in the command palette is fuzzy, so you can type part of the word to get matches. For example, you can type `trm` to search for the command "Toggle read mode".
